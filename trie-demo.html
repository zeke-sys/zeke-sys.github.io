Trie Autocomplete Demo

A browser-based version of my C++ trie autocomplete project using a sample dictionary from A to Z.

Type a prefix like ap, cyber, or sec to see autocomplete suggestions, then click a suggestion to fill the input and highlight its path in the trie diagram below.

    Trie Structure

    Under the hood

    This demo uses a trie (prefix tree) to store words character by character. Each node represents a prefix, and autocomplete suggestions are generated by walking the subtree for the prefix you type.

    The original project is written in C++, and this page mirrors the data structure in JavaScript while also rendering the trie into SVG so you can see how paths expand as you explore suggestions.
